Welcome to Scanpy Tools' documentation!
=======================================

**Scanpy Tools** is a Python library for single-cell RNA sequencing analysis
that provides various tools for preprocessing, visualization, and analysis of single-cell data.
It builds upon the `Scanpy <https://scanpy.readthedocs.io/>`_ library, and shares a *user-friendly* using the same vocabulary as scanpy API (`sctl.tl`, `sctl.pl`).

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
